# 使用ツールと事前準備

## 使用ツール一覧

### M365 Copilot — リサーチ・要件定義

- 業界背景や課題のリサーチ
- 要件定義書・議事録のドラフト生成
- 発表用ナレーション原稿の作成

### GitHub Copilot — Vibe Coding

- プロトタイプの迅速な実装
- コード補完・生成によるスピードアップ
- テストコードの生成

### Microsoft Foundry — マルチエージェント

- マルチエージェント構成の設計・実装
- エージェント間連携の構築
- プロトタイプのデプロイ

### Foundry IQ（オプション）— Agentic RAG

- ドキュメントのインデックス作成
- RAG（Retrieval-Augmented Generation）の実装
- エージェントからのナレッジ検索
