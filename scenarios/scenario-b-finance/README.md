## 3. 背景

東洋キャピタル証券のリサーチ部門は、アナリスト15名、リサーチアシスタント5名の体制で、約200社の上場企業をカバーしている。四半期ごとの決算発表シーズンには、各社の決算短信（PDF / Word）を読み込み、主要財務指標を抽出し、関連ニュースを確認し、コンプライアンスチェックを経てリサーチレポートを作成・配信している。

大手証券と比べレポート配信の速報性に課題があり（大手は数時間に対し、同社は12〜24時間）、決算集中期にはアナリストの残業が月80時間を超える状況が常態化している。情報システム部の佐藤美咲課長がリサーチ部門の要望を技術的に橋渡しする立場にあり、Azure の IaaS 利用を推進している。

## 4. 現状の課題

1. 1銘柄あたりのレポート作成に **3〜4時間**（新人は6時間以上）
2. 決算集中期（年4回）にアナリストの残業が **月80時間超**
3. レポートの品質にアナリスト間で **ばらつき** がある
4. 大手証券と比べ、レポート配信の速報性で劣る（**12〜24時間 vs 数時間**）
5. 手動のコンプライアンスチェックで **年2〜3件** の軽微な違反が発生
6. 決算短信・ニュース・コンプライアンス規程が **別々のシステムに散在**

## 8. ヒアリング時の注意事項

- **一方的に提案しない** — まずは顧客の言葉で課題を引き出すことに徹する
- **「なぜ？」を繰り返す** — 表面的な要望の裏にある真のペインを探る
- **全員が発言する** — チーム全員が役割を持ち、多角的に質問する
- **メモを取る** — 後の設計・提案に向けて、具体的な数値・発言を記録する
- **時間配分を意識** — 全ポイントをカバーできるよう計画的に進行する
